# 🎮 GameHub — Browser Arcade

> **A free, browser-based arcade with 10 games. No downloads. No ads. No tracking.**

**Created by Sai Pradeep A** · [saipradeep1126@gmail.com](mailto:saipradepp1126@gmail.com)

## 🕹️ Games Included

| Game | Type | Players |
|------|------|---------|
| 🐍 Snake | Classic | Solo |
| 🔨 Whack-a-Mole | Reflex | Solo |
| 🔢 2048 | Puzzle | Solo |
| 🧠 Memory Flip | Memory | Solo / 2P |
| ✊ Rock Paper Scissors | Strategy | Solo vs AI |
| 🏏 Cricket | Sports | Solo / 2P / vs AI |

---

## ✨ Features

- **XP & Level System** — Rookie → Legend (10 levels)
- **Achievements** — unlock badges as you play
- **Player Profiles** — local login with username/password
- **Intensity Picker** — Easy / Normal / Hard / Extreme per game
- **Smarter AI** — difficulty scales with intensity setting
- **Mobile Ready** — touch controls for all games
- **Ball-by-ball Scorecard** — cricket over history
- **Coin Toss** — animated toss for cricket 2P/AI modes
- **No Server** — runs 100% in your browser

---

## 🚀 How to Play

1. Download `index.html`
2. Open it in any browser (Chrome, Firefox, Safari, Edge)
3. That's it — no installation needed!

Or visit the live version at: https://tobirama47.github.io/Gamehub-v10/ 

---

## 📱 Mobile Support

All games work on mobile:
- **Snake** — swipe or D-pad
- **Pong** — drag finger to move paddle
- **2048** — swipe to slide tiles
- **Cricket** — tap to bat
- **Whack-a-Mole** — tap the moles!
